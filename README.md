# Agriculture Advisory Agent System

This project aims to provide automated recommendations for crop management based on weather data and pest control insights.

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/rakshithagowda-1027/rakshithagowda1027.git
   cd rakshithagowda1027
   ```
2. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```
## Usage

Run the main agriculture agent file:
```bash
python main.py
```
